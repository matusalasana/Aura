# 🛍️ AuraStore - Modern E-Commerce Experience

AuraStore is a high-performance, fully responsive e-commerce web application built with **React**. It features a seamless user journey from product discovery to checkout, emphasizing clean UI/UX and efficient state management.

## ✨ Key Features

* **Dynamic Product Catalog:** Browse products with real-time filtering and category sorting.
* **Persistent Shopping Cart:** Add, update, and remove items with totals calculated automatically.
* **Responsive Design:** Optimized for all screen sizes (Mobile, Tablet, and Desktop) using [Tailwind CSS/SASS].
* **Seamless Checkout:** Integrated [Stripe/Mock] checkout flow for a realistic shopping experience.
* **User Authentication:** Secure login and registration functionality via [Firebase/JWT].

## 🏗️ Architecture & Challenges

Building AuraStore involved several technical hurdles that sharpened my frontend skills:

1.  **State Synchronization:** Managed complex global state to ensure the cart remains consistent across different routes without page reloads.
2.  **Performance Optimization:** Implemented [Lazy Loading / Memoization] to ensure fast image rendering and smooth transitions.
3.  **Data Fetching:** Integrated [Custom API / Commerce.js] to handle asynchronous product data and error states gracefully.
